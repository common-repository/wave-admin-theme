Customize the WordPress admin theme.

== Description ==
Wave Admin Theme has the following features:

* Remove the WordPress Logo from the toolbar
* Hiding the Screen Options Tab
* Hide Admin Help Tab
* Customize the Left Footer Text
* Customize the Right Footer Text
* Remove the Admin Color Scheme Picker from the profile page

== Installation ==
1. Upload the plugin folder "wave-admin-theme" to the /wp-content/plugins/ directory.
2. Activate the plugin through the "Plugins -> Installed plugins -> Wave Admin Theme" menu in WordPress.
3. Find "Wave Admin Theme" Plugin in -> Settings options.
